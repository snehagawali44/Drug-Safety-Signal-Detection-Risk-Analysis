# Drug-Safety-Signal-Detection-Risk-Analysis
Drug Safety Signal Detection & Risk Analysis Dashboard built using Python, SQL, and Power BI on 528K+ FDA FAERS adverse event reports to identify drug safety trends, serious outcomes, and pharmacovigilance signals.

# Project Overview
This project analyzes 528,000+ FDA Adverse Event Reporting System (FAERS) records from 2015–2025 to identify drug safety signals, serious adverse outcomes, and pharmacovigilance trends.

Using Python, SQL, and Power BI, the project explores adverse drug reactions (ADRs), fatal cases, hospitalization patterns, demographic risk factors, and polypharmacy-associated safety outcomes.

# Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- FDA FAERS Dataset

# Key Features
- ADR trend and risk analysis
- Serious outcome monitoring
- Fatal and hospitalization analysis
- Top suspect drug identification
- Polypharmacy risk assessment
- Demographic-based safety insights
- Interactive Power BI dashboard

# Dashboard KPIs
- Total Reports
- Serious ADR Rate
- Hospitalized Cases
- Fatal Cases
- Critical Cases
- Recovery Rate

# Key Insights 
- Identified the most frequently reported adverse reactions.
- Analyzed fatal and life-threatening drug safety events.
- Evaluated demographic risk patterns across age groups and sex.
- Investigated polypharmacy impact on serious outcomes.
- Highlighted manufacturers and drugs associated with higher adverse event reporting.
